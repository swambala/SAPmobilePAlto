Dev Ennvironment Setup

After cloning the Project from the GitHub Make sure your Symlink the "src" folder to individual Platform(ios/android/win)

For Ex: ln -s /Users/swami/MobileApps/SAPMobilePAlto/src/ /Users/swami/MobileApps/SAPMobilePAlto/ios/www/src Note: Please use full path. Refer: http://www.tricedesigns.com/2012/02/16/linked-source-files-across-phonegap-projects-on-osx/
